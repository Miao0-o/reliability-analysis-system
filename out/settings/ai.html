返回AI 设置

SurveyLens 将统计摘要（非原始数据）从浏览器直接发送到您选择的 LLM 提供商。OpenRouter（推荐）提供多模型访问（Claude、GPT、Gemini、DeepSeek）。也支持直接使用提供商密钥，但仅限该提供商的模型。

隐私与安全
  • 原始问卷数据绝不会上传
  • 所有统计分析在浏览器本地完成
  • AI 仅接收统计摘要（约 500 字符）
  • API Key 仅存储在当前浏览器会话中
  • BYOK 模式：自有密钥，自有用量，自有账单

注意：与任何基于浏览器的 API 客户端一样，您的密钥在网络标签页中可见。这是 BYOK 架构的固有特性——不适合不受信任的共享设备。

如何获取 API Key

推荐：OpenRouter

openrouter.ai/keys → 密钥以 sk-or-v1- 开头

多模型访问（Claude、GPT、Gemini、DeepSeek）。约 $1-2 即可支持多次分析。

也支持：直接提供商密钥

sk-ant- → Anthropic (仅 Claude) · sk- → OpenAI/DeepSeek (手动选择提供商)

直接密钥在浏览器中可能有 CORS 限制。推荐使用 OpenRouter。
